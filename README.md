# GIFOS — Web App with Giphy API · JS + MediaDevices

Aplicación web desarrollada para el Segundo Parcial del Seminario de Actualización en Tecnología Web (IFTS Nº21).  
Permite buscar, grabar, generar, subir y almacenar GIFs utilizando la API pública de Giphy.

El enfoque del proyecto combina integración con Web APIs, manipulación del DOM y flujo completo de creación de contenido.

---

## Funcionalidades principales

- **Cambio de tema** con persistencia en LocalStorage  
- **Búsqueda en tiempo real** usando Giphy API  
- **Sugerencias dinámicas** y tendencias  
- **Captura de cámara** con `navigator.mediaDevices.getUserMedia()`  
- **Grabación de GIFs** usando RecordRTC  
- **Subida a Giphy** (POST) con barra de progreso  
- **Mis GIFOS** guardados en LocalStorage  
- **Responsive design** con media queries

---

## Tecnologías

- **HTML5** — estructura semántica  
- **CSS3** — estilos, responsive, temas  
- **JavaScript (ES6+)** — lógica y DOM  
- **Fetch API** — peticiones GET/POST  
- **Async/Await** — manejo asincrónico  
- **MediaDevices API** — acceso a cámara  
- **RecordRTC.js** — conversión a GIF  
- **LocalStorage** — persistencia
